GEODE-7091: Add Micrometer binders to meter registry

Add the following Micrometer binders:
- JvmGcMetrics
- ProcessorMetrics
- JvmThreadMetrics
- UptimeMetrics
- FileDescriptorMetrics

Ignore FileDescriptorMetrics binder tests on Windows
