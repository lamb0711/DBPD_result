GEODE-917: rename gemfire subprojects to geode

Rename the gemfire-* subdirectories to geode-*.  This has the effect
of changing the maven published artifacts as well.  For example,

gemfire-core-1.0.0-incubating.M2-SNAPSHOT.jar

is now

geode-core-1.0.0-incubating.M2-SNAPSHOT.jar

Also took the opportunity to rename gemfire-core-dependencies.jar
to geode-dependencies.jar.

-      final String gemfireWar = agentUtil.findWarLocation("gemfire-web");
+      final String gemfireWar = agentUtil.findWarLocation("geode-web");
-      final String pulseWar = agentUtil.findWarLocation("gemfire-pulse");
+      final String pulseWar = agentUtil.findWarLocation("geode-pulse");
-      final String gemfireAPIWar = agentUtil.findWarLocation("gemfire-web-api");
+      final String gemfireAPIWar = agentUtil.findWarLocation("geode-web-api");
