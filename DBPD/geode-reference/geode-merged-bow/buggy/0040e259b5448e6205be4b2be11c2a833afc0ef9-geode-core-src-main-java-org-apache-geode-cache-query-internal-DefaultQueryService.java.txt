GEODE-3583: Marking static variables final that are never modified

To help reduce the number of mutable statics to look through, adding
final to all of the statics that are not actually mutated outside of a
static initializer.

This change *only* consists of adding final to these variables.

-  public static boolean COPY_ON_READ_AT_ENTRY_LEVEL = Boolean
+  public static final boolean COPY_ON_READ_AT_ENTRY_LEVEL = Boolean
-  public static boolean ALLOW_UNTRUSTED_METHOD_INVOCATION = Boolean.getBoolean(
+  public static final boolean ALLOW_UNTRUSTED_METHOD_INVOCATION = Boolean.getBoolean(
